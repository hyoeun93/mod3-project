# Nature Walk Finder - GPS finder web application
Nature Walk Finder finds you hidden greenery sights in NYC for you to take a walk!!

## Features
- Users take a short quiz to find specific paths they are looking for
- Each path shows exact coordinates of the walking paths with integrated Google Maps
- Delete any path users find not matching their preferences
- Save paths they would like to add in a list

## Technologies used
- Ruby on Rails 6.0.1 (backend)
- Vanilla javascript (frontend)
- PostgresSQL 11
- Bootstrap 3
- google map api
